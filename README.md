# CommandBar Assistant

A keyboard-driven desktop assistant that helps you execute commands, automate browsing, and extract text from screenshots using OCR.
---

## 🚀 Features

- ⚡ **Global Command Bar**
  - Open using `Ctrl + \`
  - Execute commands instantly

- 🌐 **Smart Browser Automation**
  - `yt` → opens YouTube  
  - `open github`  
  - `search machine learning`

- 📸 **Screenshot + OCR**
  - Select any region on screen  
  - Extract text using Tesseract OCR  
  - Automatically copied to clipboard  

- 🗂 **Automatic Screenshot Storage**


- 🧠 **Command History**
- Shows last 2 recent commands  
- Click to autofill  

- 🔔 **System Notifications**
- Shows confirmation after extraction  

- ⚙️ **Config-based System**
- Sites stored in `config.json`  
- Easy to extend
